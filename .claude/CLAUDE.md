# Global Instructions

## 確認ルール（MUST）

ユーザーに質問・確認が必要な場合は、**必ずAskUserQuestionツールを使用すること**。

**禁止：** テキストで質問を書いて返答を待つこと
**必須：** AskUserQuestionツールで選択肢を提示すること

### AskUserQuestionを使うべき場面
- 実装方針が複数考えられる場合
- 要件に曖昧さがある場合
- ユーザーの好みを確認したい場合
- 作業を進める前に承認が必要な場合

### 例
```
❌ NG: 「どちらの方法がよいですか？」とテキストで聞く
✅ OK: AskUserQuestionツールで選択肢を提示する
```

## ブラウザ操作（MUST）

Webページの操作・確認が必要な場合は、**必ず agent-browser スキルを使用すること**。

### agent-browser を使うべき場面
- Webページのスクリーンショット取得
- フォームの入力・送信テスト
- Webアプリケーションの動作確認
- ページからのデータ抽出
- ローカルで動いているWebアプリのテスト

### 例
```
❌ NG: curlやWebFetchでHTMLを取得して推測する
✅ OK: agent-browserスキルでブラウザを操作して確認する
```
