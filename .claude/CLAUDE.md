# Global Instructions

## 確認ルール（MUST）

ユーザーに質問・確認が必要な場合は、**必ずAskUserQuestionツールを使用すること**。

**禁止：** テキストで質問を書いて返答を待つこと
**必須：** AskUserQuestionツールで選択肢を提示すること

### AskUserQuestionを使うべき場面
- 実装方針が複数考えられる場合
- 要件に曖昧さがある場合
- ユーザーの好みを確認したい場合
- 作業を進める前に承認が必要な場合

### 例
```
❌ NG: 「どちらの方法がよいですか？」とテキストで聞く
✅ OK: AskUserQuestionツールで選択肢を提示する
```
